# Quantomatic
ZX-calculus diagram to matrix plugin

## To run Quantomatic from source
with sbt installed
~~~~
$ cd [project directory]/scala
$ sbt run
~~~~

## To compile Quantomatic
with sbt installed
~~~~
$ cd [project directory]/scala
$ sbt assembly
~~~~

On linux, you can use the stript `Quantomatic.sh` to run the compiled version
